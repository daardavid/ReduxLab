# Changelog

Todas las notas de cambios importantes de ReduxLab.

## [2.0.0] - 2025-08-16
### Añadido
- Módulo Scatter Plot PCA independiente con flujo guiado.
- Etiquetas opcionales de puntos (`show_labels`).
- Porcentajes de varianza explicada en ejes y títulos de scatter.
- Archivo `THIRD_PARTY_LICENSES.txt` para transparencia de licencias.
- Auto-ejecución tras aplicar configuración del scatter.

### Cambiado
- Reescritura del diálogo de configuración del scatter para mayor robustez.
- Actualización `LICENSE.txt` completando cláusula de exoneración.
- Actualización de versión a 2.0.0 en empaquetado e instalador.

### Corregido
- Errores de indentación y referencias en la antigua implementación del diálogo scatter.

### Notas
- Se recomienda limpiar instalaciones anteriores antes de instalar 2.0.0.

## [1.0.0] - 2024-xx-xx
- Versión inicial estable.
